Work autonomously without asking questions at each step. Build, test, commit, and move to the next task.

**Why:** User wants to leave the machine running and come back to completed work. Asking for approval on every file slows things down.

**How to apply:** Follow the sprint plan, implement each task fully, run tests, commit on green, proceed to next task. Only stop to ask if genuinely blocked or a major architectural decision is needed that isn't covered by existing docs/ADRs.
